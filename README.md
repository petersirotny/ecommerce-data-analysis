# 📊 E-commerce Data Analysis (SQL + Power BI)

## 📌 Project Overview

This project analyzes e-commerce sales data using SQL and visualizes key insights in Power BI.

The goal was to identify:

* Revenue distribution by country
* Top customers by revenue
* Revenue by product category
* Top performing products

---

## 🧠 Tools Used

* SQL (data aggregation, joins, window functions)
* Power BI (dashboard & visualization)

---

## 📊 Key Insights

* Spain and Germany generate the highest revenue
* A small group of customers contributes significantly to total revenue
* Electronics and Home categories dominate sales
* Top 5 products drive a large portion of revenue

---

## 📁 Project Structure

* `/sql` → SQL queries used for analysis
* `/dashboard` → Power BI dashboard and screenshot
* `/data` → dataset
